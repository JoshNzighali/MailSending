# 📧 PHP Mail Sender with PHPMailer (Gmail SMTP)

This project demonstrates how to send emails using **PHPMailer** with **Gmail SMTP** in PHP.

It provides a secure and structured way to send HTML emails, including support for attachments, CC/BCC, and error handling.

---

## 🚀 Features

- Send emails using Gmail SMTP  
- Secure authentication with App Password  
- HTML and plain text email support  
- Error handling with exceptions  
- Support for CC, BCC, and Reply-To  
- Easy integration into any PHP project  

---

## 🛠️ Technologies Used

- **PHP (7.4+)**
- **PHPMailer**
- **Gmail SMTP**
